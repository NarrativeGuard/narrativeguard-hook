# Contributing

NarrativeGuard welcomes focused improvements that strengthen the Hook, Risk OS console, deployment flow, or documentation.

## Local Checks

```bash
npm install
npm --prefix frontend install
npm run check
```

## Pull Request Guidelines

- Keep changes scoped and explain the behavior impact.
- Add or update tests for contract logic changes.
- Do not commit `.env`, private keys, build output, dependency folders, or local deployment files.
- For frontend changes, verify the wallet flow and responsive layout before requesting review.
